Project Wrap Up and Display

Part B: Display

Reasoning for the display

My display aims to replicate the concept of the MDP around the theme of Humans versus Robots, which is best accomplished by presenting the finish work in its environment on a computer system.

The display aims to do at least the following:

  • Convey the MDP directly to the marker by allowing them to engage with it by playing it.
  • Provide a suitable space for the marker to view the media content of the folio and the folio in digital form.
  • Allow the marker to view the git log and source code of the project to validate the authenticity of my work

Needs for the display

The display needs the following items:

  • Computer desk
  • The following computer peripherals and connectors:
    • Monitor
    • VGA cable
    • Mouse
    • Keyboard
  • Computer for which I have administrative privileges and the following:
    • Steam client (this is a prerequisite for Garry's Mod)
    • Garry's Mod
    • Project Folio in digital form, to be displayed in the web browser
  • Printed copy of the folio in case the digital folio fails

Evaluation procedure for the display

The display will be evaluated based on how it meets the needs of the design as well as its function and aesthetics.

Area of Evaluation Criterion for evaluation
Needs Does the display allow the marker to play HvR in an enjoyable way?
Needs Is the display space suitable for the marker to view the digital MDP folio?
Needs Does the display easily allow the marker to validate the authenticity of the work?
Function Does the digital folio have no broken links or missing images?
Function Does the display system have a functional copy of HvR?
Function Does the display easily allow for the marker to view the git log and source code to validate the authenticity of the MDP?
Aesthetics Is the display aesthetically pleasing?
Aesthetics Is the display appropriate to the theme and colours of HvR?
